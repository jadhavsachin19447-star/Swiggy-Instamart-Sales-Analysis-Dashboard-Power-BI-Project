# Swiggy-Instamart-Sales-Analysis-Dashboard-Power-BI-Project
I created a sales analysis dashboard for Swiggy Instamart using Power BI. In this project, I analyzed total sales, orders, and customer ratings. I used Power Query for data cleaning and DAX for calculations like total sales and average sales. This project helped me understand how to convert raw data into meaningful business insights.
